## 离线语音交互控制优势

传统家电控制方式有以下缺点：
- 按键控制：需要走近产品，按键复杂，不方便老人和小孩使用；
- 遥控控制：找不到遥控器，并且容易丢失和损坏；
- APP控制：需要给设备配网，每次控制都需要打开APP；
- 智能音箱控制：需要给设备配网，依赖智能音箱，响应慢，不保护隐私。

:::info
使用离线语音交互方案可以给用户带来卓越的综合用户体验，是极具性价比的智能解决方案。
:::
